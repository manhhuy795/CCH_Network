# Legacy SDN Demo

Demo chinh thuc dung cho bao ve la:

```text
sdn_mpls_demo/
```

Khong dung `sdn_demo/` trong buoi bao ve chinh. `sdn_demo/` chi mo phong SDN toi gian voi mot OVS trung tam va controller standalone, khong phai mo hinh "Hybrid MPLS L3VPN Logic Simulation + SDN Edge Policy for Call Center BPO" hien tai.
